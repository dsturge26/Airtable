How I did it · AI fluency, not AI theater

The method behind the number

105,508 free-form prompts is unstructured chaos. Turning it into a board-ready value story took a repeatable AI pipeline — the same muscle I'd bring to standing up a transformation methodology and adoption playbooks.

1

Semantic categorization at scale

Used LLM classification to map every prompt — including 9,110 free-form requests with no shortcut — to a workflow and purpose (e.g. "analyze risk for Deal ID 71228656" → Deal Pulse). Result: 89.5% categorized into renewal activities.

2

Complexity-tiered time modeling

Classified non-shortcut prompts into Simple / Moderate / Complex and assigned conservative, defensible time-equivalents (3–5, 10–20, 30–60 min) rather than one blanket assumption.

3

Two independent evidence tracks

Built seller-reported actuals and the model separately, so the headline number is corroborated, not asserted — the way you earn an executive's trust in an AI claim.

4

Adoption + value tiering

Segmented every role into Inactive → Active → Engaged → Power on both usage and hours banked, turning a flat "we rolled it out" into a targeted "here's who's next" plan.
